# Database Schema

## Collections

### Users
```
{
  _id: ObjectId,
  email: string (unique),
  name: string,
  password: string (hashed, optional for OAuth),
  googleId: string (unique, optional),
  googleCalendarId: string,
  googleRefreshToken: string,
  createdAt: Date,
  updatedAt: Date
}
```

### Trips
```
{
  _id: ObjectId,
  userId: ObjectId,
  name: string,
  description: string,
  startDate: Date,
  endDate: Date,
  countries: [string],
  cities: [{
    _id: ObjectId,
    name: string,
    country: string,
    arrivalDate: Date,
    departureDate: Date,
    notes: string
  }],
  createdAt: Date,
  updatedAt: Date
}
```

### Flights
```
{
  _id: ObjectId,
  tripId: ObjectId,
  airline: string,
  flightNumber: string,
  departureAirport: string,
  arrivalAirport: string,
  departureTime: Date,
  arrivalTime: Date,
  notes: string,
  createdAt: Date,
  updatedAt: Date
}
```

### Accommodations
```
{
  _id: ObjectId,
  tripId: ObjectId,
  hotelName: string,
  city: string,
  checkInDate: Date,
  checkOutDate: Date,
  address: string,
  confirmationNumber: string,
  notes: string,
  bookingLink: string,
  createdAt: Date,
  updatedAt: Date
}
```

### Transportation
```
{
  _id: ObjectId,
  tripId: ObjectId,
  fromCity: string,
  toCity: string,
  type: enum ['train', 'bus', 'car', 'ferry', 'other'],
  departureTime: Date,
  arrivalTime: Date,
  notes: string,
  createdAt: Date,
  updatedAt: Date
}
```

### Activities
```
{
  _id: ObjectId,
  tripId: ObjectId,
  title: string,
  city: string,
  date: Date,
  type: enum ['museum', 'restaurant', 'tour', 'hike', 'other'],
  notes: string,
  bookingLink: string,
  coordinates: {
    lat: number,
    lng: number
  },
  createdAt: Date,
  updatedAt: Date
}
```

### Tasks
```
{
  _id: ObjectId,
  tripId: ObjectId,
  title: string,
  description: string,
  dueDate: Date,
  completed: boolean,
  category: enum ['packing', 'booking', 'preparation', 'general'],
  isAutoGenerated: boolean,
  createdAt: Date,
  updatedAt: Date
}
```

## Indexes

- `user`: userId
- `trips`: userId, startDate
- `flights`: tripId, departureTime
- `accommodations`: tripId, checkInDate
- `transportations`: tripId, departureTime
- `activities`: tripId, date
- `tasks`: tripId, completed, dueDate
